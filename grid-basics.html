Basics

CSS Grid is one of the best options for layout. It is both versatile and flexible, while also providing for consistency and precision when needed.

Just like flexbox, we start with the parent element, applying display: grid; as well as a combination grid-template-columns and other sizing and layout properties to set the initial layout. We then target the child elements and apply properties like grid-column, grid-row and others to adjust their placement and alignment on the grid.
